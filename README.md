# Classification-Rock-Paper-Scissors
Implementasi CNN dengan TensorFlow/Keras untuk klasifikasi gambar gestur tangan rock, paper, scissors. Program secara otomatis mengunduh dataset, membagi data menjadi train/validation, dan membangun pipeline ImageDataGenerator dengan augmentasi (rotasi, shift, shear, zoom, flip) untuk meningkatkan generalisasi model. Arsitektur model berupa jaringan konvolusional berurutan dengan beberapa blok Conv2D–MaxPooling, diikuti Flatten, Dense 512 (ReLU), dan output Softmax 3 kelas. Training dijalankan hingga 25 epoch dengan callback pembatas waktu (±30 menit), kemudian dievaluasi pada data validasi. Tersedia juga fungsi pada Google Colab untuk mengunggah gambar baru dan memprediksi kelasnya secara langsung.

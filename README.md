# Classification-Rock-Paper-Scissors
Implementasi CNN berbasis TensorFlow/Keras untuk klasifikasi citra gestur rock–paper–scissors. Mencakup otomatisasi unduh dataset, pembagian train/validation, augmentasi dengan ImageDataGenerator, arsitektur Conv2D–MaxPooling–Softmax (3 kelas), training ±25 epoch, evaluasi, dan prediksi gambar baru via Colab.
